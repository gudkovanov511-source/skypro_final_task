Git - всему голова!

  • ls — отображает список файлов в текущем каталоге.
  • cd - расшифровывается как "Change directory", что переводится как "сменить папку" или "перейти в другую папку".
  • pwd - расшифровывается как "Print working directory", что переводится как "распечатать рабочий каталог". В контексте Git это команда, которая выводит путь к текущему рабочему каталогу (директории), в котором находится пользователь в данный момент.
  • mkdir - расшифровывается как "Make directory", что переводится как "создать папку" и может использоваться для создания структуры директорий в локальной рабочей области репозитория.
  • git mv - команда, с помощью которой можно переместить файл.
